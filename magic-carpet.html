Magic Carpet

An Automatic Adventure App

By: Jude Dutton, Zach Thomas, and Haley Mesander

Summary

Magic carpet is a mobile-first application that solves one problem: "Where should we go to eat?". Magic Carpet will randomly select a nearby restaurant for you and, keeping the choice of destination a surprise, call a ride for you to get there.

Right now, the application is a web app designed to appear on mobile devices specifically, but can be used on laptop and desktop computers as well. The app can also cancel the ride for you if necessary

Future Iterations

Creating a settings panel is currently a work-in-progress. We would like for the user to be able to adjust how far they would like to search, how much money they would like to spend when they get there, and choose the rating-level for their destination.

Magic Carpet will enhance its ride experience by keeping track of the ride share service's progress, as well as showing you reviews for your destination while you are in the ride.

Magic Carpet will enhance its feature set by adding the ability to choose other events and places beyond restaurants.

Our Tech Stack

We are currently powering our app with the Yelp API for restaurant choices and the Lyft API for ride share services. We've built an internal API using Rails that is the hub for our requests and potential future hub for machine learning. Our frontend interface is built with React
